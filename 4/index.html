Common Block Elements:

- Always start on a new line.
- Take up the full available width (even if the content is smaller).
- Can contain inline and sometimes other block elements.

div → generic container
p → paragraph
h1 → headings
ul, ol, li → lists
table, tr, th, td → tables
section, article, header, footer, nav → semantic containers
form → form container
blockquote → block quotations

Inline Elements

- Do not start on a new line.
- Only take up as much width as their content needs.
- Can only contain text or other inline elements (generally, not block).

span → generic inline container a → hyperlink strong → bold text (semantic) em → italic/emphasis b and i → styling (non-semantic) img → image br → line break label → form label input, textarea, select → form controls code → inline code small → smaller text
